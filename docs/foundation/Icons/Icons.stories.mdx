# Icons

You can import our React icons from the icon library as follows:

```ts
import { ActualSize } from "@hitachivantara/uikit-react-icons";
```

And you can use them like so:

```tsx
<ActualSize />
```

This final result will be:

  <ActualSize />
</Preview>

# Pictograms library

You can import our React pictograms from the icon library as follows:

```ts
import { FemaleRunning } from "@hitachivantara/uikit-react-icons";
```

And you can use them like so:

```tsx
<FemaleRunning />
```
